Setup
1. Go to c:/
2. Type   git config --global user.name "Your name"
3. Type   git config --global user.email "Your github email address"
4. Type    git clone https://github.com/ahinsutime/Rproject.git




Useful git commands


git status               :                                    See the current status of repository

git add -A              :                                    Add (Staging) all modified files

git commit -m "any message you want"     :       Get ready for upload

git push                 :      Uploading your modifications to online repository

git pull                   :      Upating your local git repository

git checkout --ours CONFLICTED_FILE_NAME   :  Resolving conflict with my code

git checkout --theris COMFLICTED_FILE_NAME  : Resolving conflict with their code
